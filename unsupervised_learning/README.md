# Unsupervised Learning & Dimensionality Reduction

k-means and EM clustering, plus PCA / ICA / random projection / feature selection, evaluated on Breast Cancer and Titanic feature spaces.

## Business framing

Segmentation and compression are prerequisites for targeting and visualization when labels are scarce. This module compares how DR choices change cluster structure and downstream supervised utility.

## How to run

```bash
jupyter notebook unsupervised_learning/notebooks/unsupervised_dimensionality.ipynb
```

Analysis notes: [`docs/analysis.md`](docs/analysis.md). Cluster center exports (when generated) live under [`assets/`](assets/).
