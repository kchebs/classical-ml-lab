# Sequential Decisions (MDPs & Q-Learning)

Value iteration, policy iteration, and Q-learning on Frozen Lake and Forest-style MDPs — comparing planning vs learning and sensitivity to γ, α, and episode count.

## Business framing

Sequential decisions under uncertainty appear in inventory, routing, and interactive products. This module is a **methods lab** for classical MDP solvers and tabular Q-learning (not a live trading system).

## How to run

```bash
# from classical-ml-lab root; full suite can take a long time
python sequential_decisions/run.py
```

Committed figures and CSV scorecards: [`assets/`](assets/). Analysis notes: [`docs/analysis.md`](docs/analysis.md).
